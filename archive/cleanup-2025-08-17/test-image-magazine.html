🖼️ Image Magazine Feature Test

📋 Test Overview

This test verifies the complete Image Magazine functionality for the editor, including:

  • Images Button: IMAGES button appears in sidebar next to CATEGORY
  • Magazine Toggle: Clicking opens/closes draggable image magazine
  • Draggable Window: Box Style 1 frame can be repositioned around page
  • Image Gallery: Shows available images in scrollable grid
  • Import Button: Allows uploading images to GitHub repository
  • Drag & Drop: Images can be dragged into visual editor
  • File Drop Zone: Visual editor accepts dropped image files
  • Visual Preview: Images appear with clear visual preview showing exactly where they'll be in post
  • Image Controls: Hover to see edit/remove buttons for each image
  • Markdown Export: Images convert to proper markdown format when exporting
